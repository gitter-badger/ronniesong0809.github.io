Linked List Cycle

Posted on 2018-11-08 | In LeetCode | | Views:

Given a linked list, determine if it has a cycle in it.

Follow up:
Can you solve it without using extra space?

The way to solve this problem is to set two pointers, slow and fast from the head of linked list. By moving slow pointer 1-node/time and fast pointer 2-nodes/time. If there is a loop, then they will definitely meet.
